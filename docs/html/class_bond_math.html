Detailed Description

A static class for calculating bond price and yield. Validation of input is generally left to the caller which is from Qt line edit input validation in this case. Very large values will cause low accurcy results.

Calculates the bond yield using an iterative approach. Approaches the correct yield by adding/subtracting to the rate based on the difference to the correct price by an amount that gets smaller.
